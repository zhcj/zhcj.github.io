SVN版本控制系统安装使用手册（网站开发用）

一、Linux服务器端

1、安装subversion

yum install subversion -y

2、创建SVN空版本库

mkdir /myweb
svnadmin create /myweb/myrepo

3、配置SVN

未完成，待续。。。

4、创建或迁移网站目录

svn co file:///myweb/myrepo /myweb/www
cp /原代码目录/* /myweb/www/
cd /myweb/www
svn add *
svn ci -m "创建项目"

5、设置网站钩子（提交代码到SVN版本库后，自动拉取到网站目录）

cd /myweb/myrepo/hooks
echo "#!/bin/sh" > post-commit
echo "cd /myweb/www" >> post-commit
echo "/usr/bin/svn up" >> post-commit
echo "chown www:www /myweb/www -R >> post-commit
chmod +x post-commit
