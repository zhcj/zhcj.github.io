Squid安装配置笔记

注：网站系统为CentOS 7/8，使用systemd。

一、安装依赖

yum install perl gcc autoconf automake make sudo wget libxml2-devel libcap-devel -y

二、下载源码、编译安装

1、下载

wget http://www.squid-cache.org/Versions/v4/squid-4.9.tar.xz

2、解压、编译

tar xvf squid-4.9.tar.xz
cd squid-4.9
./configure --prefix=/data/squid
make -j5
make install
