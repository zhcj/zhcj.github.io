RoketMQ安装配置笔记

注：网站系统为Linux系统，使用systemd。

一、安装依赖：

1、安装配置JDK

下载地址（需要登录Oracle）：

https://www.oracle.com/java/technologies/javase-downloads.html

配置（以jdk-8u251-linux-x64.tar.gz为例）：

tar xvf jdk-8u251-linux-x64.tar.gz -C /data/mq/
echo "export JAVA_HOME=/data/mq/jdk1.8.0_251" >> /etc/profile
echo "export CLASSPATH=.:\${JAVA_HOME}/jre/lib/rt.jar:\${JAVA_HOME}/lib/dt.jar:\${JAVA_HOME}/lib/tools.jar" >> /etc/profile
echo "export PATH=\$PATH:\${JAVA_HOME}/bin" >> /etc/profile
source /etc/profile

2、安装配置Maven：

wget https://archive.apache.org/dist/maven/maven-3/3.2.5/binaries/apache-maven-3.2.5-bin.tar.gz
tar xvf apache-maven-3.2.5-bin.tar.gz -C /data/mq/
echo "export PATH=/data/mq/apache-maven-3.2.5/bin:\$PATH" >> /etc/profile
source /etc/profile

mvn -v确认一下，出现以下类似信息，说明环境配置成功正确：

Apache Maven 3.2.5 (12a6b3acb947671f09b81f49094c53f426d8cea1; 2014-12-15T01:29:23+08:00)
Maven home: /data/mq/apache-maven-3.2.5
Java version: 1.8.0_251, vendor: Oracle Corporation
Java home: /data/mq/jdk1.8.0_251/jre
Default locale: zh_CN, platform encoding: UTF-8
OS name: "linux", version: "3.10.0-1062.12.1.el7.x86_64", arch: "amd64", family: "unix"

注：为加快包的下载速度，建议配置aliyun镜像：将以下代码加入到conf/settings.xml文件的标签中作为子节点（加在下面）

<mirror>
    <id>aliyunmaven</id>
    <mirrorOf>*</mirrorOf>
    <name>阿里云公共仓库</name>
    <url>https://maven.aliyun.com/repository/public</url>
</mirror>

二、下载并安装RocketMQ：

wget https://archive.apache.org/dist/rocketmq/4.7.0/rocketmq-all-4.7.0-source-release.zip
unzip rocketmq-all-4.7.0-source-release.zip -d /data/mq/
cd rocketmq-all-4.7.0-source-release
mvn -Prelease-all -DskipTests clean install -U
#上条命令会下载大量的包到~/.m2/repository目录，建议备份此目录
cd distribution/target/rocketmq-4.7.0/rocketmq-4.7.0
#启动Name Server
nohup sh bin/mqnamesrv &
#查看
tail -f ~/logs/rocketmqlogs/namesrv.log
#启动Broker
nohup sh bin/mqbroker -n localhost:9876 &
#查看
tail -f ~/logs/rocketmqlogs/broker.log
