VNC server安装使用手册

一、服务器端安装

1、安装tigervnc-server（前提：必须安装X界面，建议加epel源，安装xfce环境：yum groupinstall xfce）

yum install tigervnc-server -y

2、创建vnc登录密码

vncpasswd

3、修改systemd启动项中的用户名为root（建议非root用户）

sed -i "s//root/g" /usr/lib/systemd/system/vncserver@.service

4、打开Firewalld防火墙

firewall-cmd --permanent --add-port=5901/tcp
firewall-cmd --reload

5、修改配置

sed -i "s/\$geometry = \"1024x768\";/\$geometry = \"800x600\";/g" /usr/bin/vncserver

6、启动vncserver

systemctl start vncserver@:1

二、客户端用5901端口连接，输入密码即可
